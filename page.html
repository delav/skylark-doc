导航菜单 ​

一个基础的导航，首页、文章路径以及外站链接

说明

text是导航中显示的文本

ts
export default defineConfig({

  themeConfig: {
    //导航栏
    nav: [
      { text: '首页', link: '/' }, 
      { text: '快速上手', link: '/getting-started' },
